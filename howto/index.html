How to use the Lynxis Interface

  • Press 'tab' to switch between edit and view mode
  • In view mode:
    • Click a node to load that node's relation page
    • Press 'q' to load the page for the root node
    • Press 'w' to input a node id to load the page for
  • In edit mode:
    • Click a node to de-parse MathJax and edit the node's text
    • Press 'backspace' with caret at beginning of node to delete node
    • Press 'enter' to create a new node with relationship 'contains' to root node at end of document
