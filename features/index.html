Features of the Lynxis Interface

  • Press 'tab' to switch between edit and view mode
  • MathJax Parsing:
    • Any TeX put between dollar signs will be parsed using the MathJax API on both page reload or selection of a different node.
    • To edit a node containing TeX mathematics, simply click the node while in edit mode and the text will be de-parsed. Then, edit or paste new text.
  • Editing built around Contenteditable API
